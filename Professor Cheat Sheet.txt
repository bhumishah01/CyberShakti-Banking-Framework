RURALSHIELD – PROFESSOR CHEAT SHEET (1 PAGE)

1) PROBLEM STATEMENT (ONE LINE)
Cybersecurity Framework for Rural Digital Banking (SIH PSID 25205): offline‑first, fraud detection + secure authentication for low‑end phones and weak internet.

2) CORE IDEA (ONE LINE)
Store locally first → detect fraud locally → sync later when network is safe.

3) WHAT I BUILT (KEY FEATURES)
- Local encrypted storage in SQLite
- Fraud scoring rules (new recipient, high amount, odd hour, rapid burst, failed logins)
- Actions: allow / hold / block
- Trusted contact approvals for risky cases
- Audit log + change log (old/new values)
- Offline outbox + sync queue + night sync simulator
- Multilingual UI + voice safety prompt
